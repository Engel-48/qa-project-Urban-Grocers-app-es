# Proyecto Urban Grocers
En este proyecto se ejecutan 9 pruebas creando un nuevo
usuario y a la vez creando un kit. Con la peculiaridad 
de cambiar el nombre del kit según sea la lista de pruebas
dando como resultado si pasaron o fallaron.

Se usa como fuente de documentación los requisitos del servidor
terminado en '/docs/'. 
    Crear usuario
    Crear nuevo kit

Se utilizaron los paquetes de 'requets' y 'pytest' para las pruebas.
